# Talent IQ
AI-Enabled Intelligent Assessment & Hiring Platform

## Overview
Talent IQ is an AI-based hiring platform that analyzes job descriptions
and automatically generates skill-based assessments.
It evaluates candidates using AI-driven scoring to ensure fair,
transparent and skill-based hiring.

## Features
- Recruiter Dashboard
- Candidate Dashboard
- JD-Based Skill Assessment
- AI Evaluation (Demo)
- Result & Ranking System
- Live Leaderboard

## Technology Used
- HTML
- CSS
- JavaScript (Basic)

## How It Works
1. Recruiter uploads Job Description
2. System extracts required skills
3. Assessment is generated automatically
4. Candidates attempt the test
5. AI evaluates answers
6. Results & ranking are displayed
